# 阶段 1 验收指标

阶段 1 定义指标、计算和未来门槛；除静态场景目录与现有离线回归外，不声称已测量语音、电话、POS、人工或生产指标。百分比目标按一个发布候选的同一版本和固定测试集计算，分母为 0 时指标记为 `NOT_MEASURED`，不得记为通过。

## 3. 阶段 1 可测与不可测

当前可测：场景目录结构和静态覆盖、四语言数量、分类不变量、引用完整性、离线 guard、现有 pytest/Vitest/TypeScript/build、V3 false mutation/confirmation bypass/live LLM trigger。

当前不可测：真实三语识别准确率、人工接管运营指标、电话和媒体延迟、商家/POS 状态准确率、顾客完成率及所有真实成本。不可测指标保留明确未来阶段、数据源和暂定门槛；不得用模拟通过替代生产结论。
